# falconsocial

## API
* ```GET /api/items/```
* ```PUT /api/items/``` => ```items:put(ids)```
* ```POST /api/items/``` => ```items:post(id)```
* ```DELETE /api/items/``` => ```items:delete:all()```
* ```GET /api/items/:id```
* ```PUT /api/items/:id``` => ```items:id:put(item)```, ```items:post(id)```
* ```DELETE /api/items/:id``` => ```items:delete(id)```
